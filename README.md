# FreeAPI Quotes App

A minimal quotes browsing and copying app built with vanilla JavaScript.

## Features

- Quote of the Day (random, highlighted)
- Scrollable quotes feed
- Search by author or content
- Copy quote to clipboard
- Load more pagination
- Loading skeletons & error handling

## Tech Stack

- HTML, CSS, JavaScript (no frameworks)
- Tailwind CSS (CDN)
- FreeAPI Quotes endpoint

## API Endpoint Used

```
GET https://api.freeapi.app/api/v1/public/quotes
```
